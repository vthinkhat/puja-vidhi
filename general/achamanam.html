ಆಚಮನಂ (Achamanam)

For the first three verses, sip water and recite, ending with `svaha` for males and `namah` for females. Then, wash your hands.

ಓಂ ಕೇಶವಾಯ ಸ್ವಾಹಾ → ಓಂ ನಾರಾಯಣಾಯ ಸ್ವಾಹಾ → ಓಂ ಮಾಧವಾಯ ಸ್ವಾಹಾ
Om Keshavaya Svaha → Om Narayanaya Svaha → Om Madhavaya Svaha
 ಗೋವಿಂದಾಯ ನಮಃ → ವಿಷ್ಣವೇ ನಮಃ → ಮಧುಸೂದನಾಯ ನಮಃ
Govindaaya Namaha → Vishnave Namaha → Madhusuudanaaya Namaha
 ತ್ರಿವಿಕ್ರಮಾಯ ನಮಃ → ವಾಮನಾಯ ನಮಃ → ಶ್ರೀಧರಾಯ ನಮಃ
Trivikramaaya Namaha → Vaamanaaya Namaha → Sridharaaya Namaha
 ಹೃಷೀಕೇಶಾಯ ನಮಃ → ಪದ್ಮನಾಭಾಯ ನಮಃ → ದಾಮೋದರಾಯ ನಮಃ
Hrishikeshaaya Namaha → Padmanaabhaaya Namaha → Daamoodaraaya Namaha
 ಸಂಕರ್ಷಣಾಯ ನಮಃ → ವಾಸುದೇವಾಯ ನಮಃ → ಪ್ರದ್ಯುಮ್ನಾಯ ನಮಃ
Sankarshanaaya Namaha → Vasudevaaya Namaha → Pradyumnaaya Namaha
 ಅನಿರುದ್ಧಾಯ ನಮಃ → ಪುರುಷೋತ್ತಮಾಯ ನಮಃ → ಅಧೋಕ್ಷಜಾಯ ನಮಃ
Aniruddhaaya Namaha → Purushottamaaya Namaha → Adhokshajaaya Namaha
 ನಾರಸಿಂಹಾಯ ನಮಃ → ಅಚ್ಯುತಾಯ ನಮಃ → ಜನಾರ್ದನಾಯ ನಮಃ
Narasimhaaya Namaha → Achyutaaya Namaha → Jaanardanaaya Namaha
 ಉಪೇಂದ್ರಾಯ ನಮಃ → ಹರಯೇ ನಮಃ → ಶ್ರೀಕೃಷ್ಣಾಯ ನಮಃ
Upendraaya Namaha → Haraye Namaha → Srikrishnaaya Namaha
